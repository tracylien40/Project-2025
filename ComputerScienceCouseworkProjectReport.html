Plan & Design

The Interactive Crypto Information System  will serve as an online tool that:

  • Collects Data: Retrieves real-time and historical cryptocurrency data from public Data from (Kaggle).
  • Processes Data: Cleans and organizes the raw data to ensure accuracy.
  • Analyzes Data: Calculates key market metrics such as price changes, trading volumes, and overall market trends.
  • Visualises Data: Provides interactive charts and graphs for user-friendly data interpretation.
  • Allows Interaction: Allows users to filter and compare information easily using a interactive web interface built with HTML , while Python handles the data training and processing.
  • Objective: To retrieve cryptocurrency data from a  public source and display data in an interactive website

Functionality:

  •  A Python script connects to Bitcoin 2 SV.csv file to fetch data on prices, volumes, and market capitalisation.
  • Data Storage: The retrieved data is saved in a structured CSV format  for later processing.

 Data Analysis and Visualization

  • Visualisation: Charts and graphs (line, bar, and pie charts) are generated using libraries like Matplotlib or Plotly.
  • Interactivity: Visualizations allow users to hover for details or click to filter specific data segments when trading, buying or selling.
